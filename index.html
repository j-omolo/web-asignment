About the Box Model

The CSS Box Model is a fundamental concept that describes how elements are rendered on a webpage.

Every HTML element is treated as a box with the following properties:

  • Content - The actual content of the element
  • Padding - Space between the content and the border
  • Border - A line surrounding the padding
  • Margin - Space outside the border

Box Model Features

Content

The innermost part of the box where text and images appear.

Padding

Creates space around the content, inside the border.

Border

A line that surrounds the padding and content.

Margin

Creates space around the element, outside the border.
